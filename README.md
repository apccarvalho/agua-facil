# 💧 ÁguaFácil
 
Permite que clientes visualizem faturas e histórico de consumo, funcionários gerenciem contas e leitores registrem leituras de hidrômetro — tudo em uma interface intuitiva.
 
---

## 📋 Sobre o projeto
 
O **ÁguaFácil** centraliza as informações da conta de água em um sistema web simples, com acesso diferenciado por perfil de usuário.
 
Projeto acadêmico desenvolvido para a disciplina de Web II — IFTM.
 
---

## ✅ Funcionalidades
 
- Autenticação com controle de acesso por perfil
- **Cliente:** visualiza faturas, histórico de leituras e consumo por imóvel
- **Funcionário:** acessa e gerencia dados de qualquer cliente
- **Leitor:** registra leituras de hidrômetro vinculadas ao imóvel
- Cálculo de consumo com faixas tarifárias
- Status de fatura: Aberta, Paga e Vencida

---

## 🛠️ Tecnologias
 
- **Java 21**
- **Spring Boot 3**
- **Spring Security** — autenticação e controle de acesso por perfil
- **Spring Data JPA** — mapeamento objeto-relacional
- **Thymeleaf** — renderização de páginas no servidor
- **MySQL** — banco de dados
- **Maven** — gerenciamento de dependências
